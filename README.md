# SwiftLightBlue

Daisuke Bekki氏の日本語CCGパーザである[lightblue](https://github.com/DaisukeBekki/lightblue)をSwiftに移植するものです。現在開発中です。

元のソースコードに忠実に移植するため、機能や実装はほとんど変更はしていません。

また、現在以下は未実装です。
* DTSの処理
* Visualizerの処理
